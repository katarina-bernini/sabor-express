# Sabor Express 

O **Sabor Express** é uma aplicação prática desenvolvida em Python com o objetivo de gerenciar o cadastro, 
ativação e avaliação de restaurantes. O projeto simula o funcionamento lógico de uma plataforma de gerenciamento 
de estabelecimentos comerciais via terminal.

## Funcionalidades do Sistema
- **Cadastro de Restaurantes:** Permite adicionar novos estabelecimentos ao sistema de forma dinâmica.
- **Alternar Estado (Ativação):** Ativa ou desativa o status de funcionamento de um restaurante.
- **Listagem Estruturada:** Exibe na tela todos os restaurantes cadastrados com suas respectivas categorias e status atuais.

## Conceitos Técnicos Aplicados
- **Estruturas de Dados:** Manipulação avançada de listas e dicionários para armazenamento temporário dos dados.
- **Modularização e Funções:** Organização do código em blocos de funções limpas e reutilizáveis.
- **Lógica de Controle e Loops:** Uso de estruturas condicionais (`if/else`) e laços de repetição (`for`/`while`) para navegação no menu interativo.
- **Tratamento de Exceções:** Implementação de tratamento de erros básicos para entradas inválidas do usuário (ex: `try/except`).

## Tecnologias Utilizadas
- **Linguagem:** Python
- **Ambiente de Desenvolvimento:** VS Code
